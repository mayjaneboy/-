元素显示模式就是元素（标签）以什么模式显示。一般分为块元素和行内元素两种

块元素

块元素(p、h、div、ul、ol、li)独占一行，高度、宽度、边距都可以设置，是一个容器或盒子，里面可以放行内元素和块元素

块元素的宽度默认为父级标签的宽度。文字类的块元素(p、h)内不能再放块元素

行内元素

行内元素 在一行上可以显示多个 高、宽不能单独设置 默认宽度为其本身宽度
行内元素(除a标签)不能容纳块元素 链接(a标签)里面不能再放链接

行内块元素

行内块元素(img input td)，一行可以有多个
默认宽度为其本身宽度
高、宽、边距可以设置
个人主页(display: block;将行内元素转换成块元素，可以设置高、宽、颜色)
display:inline;将块元素转换为行内元素(这样就不能设置高、宽)

display: inline-block; 行内元素转换为行内块元素 就可以设置高、宽 一行上也可以显示多个元素
