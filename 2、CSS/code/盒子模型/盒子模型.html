外边距塌陷问题

子块未能正确向下移20px

在父块内设置 overflow: hidden; 后

子块正确向下移20px

实际我觉得，外边距的意思是 被上一个块元素的最下面位置排斥一个外边距的距离。 而嵌套的子块仍是相对于位置上的上一个块元素来被排斥的，而不是相对于父块。 而设置了 overflow: hidden; 后，就变成相当于父块。
